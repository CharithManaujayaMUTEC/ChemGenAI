# ChemGenAI

AI-powered molecular generation platform using:

- Variational Autoencoders (VAE)
- SMILES molecular representation
- PyTorch
- FastAPI
- MLflow
- Docker
- MLOps pipeline

## Features

- Generate novel molecular structures
- REST API with FastAPI
- Containerized deployment
- MLflow experiment tracking
- Hugging Face Spaces deployment

## Project Structure

```text
src/
├── api/
├── data/
└── models/

models/
├── lstm_vae.pt

```
